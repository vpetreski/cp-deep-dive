# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

The full text is maintained upstream at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
and is adopted here by reference.

## Scope

This Code applies within all project spaces (issues, pull requests, discussions,
code review, documentation, public presentations that reference the project) and
when representing the project in public venues.

## Reporting

If you witness or experience conduct that appears to violate the Covenant,
please report it privately by emailing:

> **vanja@petreski.co**

Reports are handled confidentially. We aim to acknowledge every report within
three working days and respond with next steps within ten working days.

## Enforcement

The project maintainer reviews reports and decides on the response. Possible
outcomes, in escalating order, follow the Covenant's enforcement guidelines:

1. Private correction
2. Warning
3. Temporary ban from project spaces
4. Permanent ban from project spaces
